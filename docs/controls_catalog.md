# Controls Catalog

Placeholder catalog for security controls implemented by the checker modules.

Current milestone: **Milestone 1 — Repository Foundation and Security Engineering Scaffold**.

Controls are documented as placeholders only. The MVP control logic is planned for Milestone 5 after configuration, client, and collector layers are in place.

Each control should include:

- Control ID
- Objective
- Data sources
- Severity
- Detection logic
- Remediation guidance
- Compliance mappings
